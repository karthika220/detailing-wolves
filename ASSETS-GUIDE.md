## üé® Typography Setup

### Google Fonts Integration

Add this to your `<head>` section in index.html (or before `styles.css`):

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700;800&family=Poppins:wght@400;500;600;700;800&display=swap" rel="stylesheet">
```

### Font Usage:

**Montserrat** (Primary):
- Headlines (H1, H2, H3)
- Buttons
- Navigation

**Poppins** (Alternative):
- Body text
- Descriptions
- Form inputs

### Font Weights:
- 400 - Regular (body text)
- 500 - Medium (subheadings)
- 600 - Semi-bold (card titles)
- 700 - Bold (section headings)
- 800 - Extra-bold (hero title)

## üìè Image Optimization Tips

1. **Compress all images** before uploading
   - JPG: Use 80-85% quality
   - PNG: Use compression tools
   - Aim for under 200KB per image

2. **Correct dimensions**:
   - Don't upload larger than needed
   - Use exact sizes specified above
   - Maintain aspect ratios

3. **File naming**:
   - Use lowercase
   - No spaces (use hyphens)
   - Descriptive names

4. **Format choice**:
   - JPG: Photos, complex images
   - PNG: Logos, icons (need transparency)
   - SVG: Icons, simple graphics (best for icons)
